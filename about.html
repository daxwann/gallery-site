About us

Website designed and developed by Xiaoda Wang

In 2017, I established the fictional gallery for my web development class at Los Angeles City College. For the final project of this course, I designed and built this website to showcase some of my skills in HTML5/CSS3 and JavaScript.

Inspired by the minimalist web designs of MoMA and the Guggenheim, I created a simple, two-column website with a header and footer. I used a high-contrast, monochromatic scheme to reflect the principles of minimalism. For typography, I utilized the Bauhaus style of Archivo typeface from Google fonts, which is reminiscent of the typeface Franklin Gothic that inspired MoMa's own typeface.

I utilized various methods to create spatial balance and to make the website more responsive. CSS grids were useful to divide the page into balanced sections based on fractions or fixed units. The size of the sections will adjust accordingly to the size of the window. Another way to adjust the structure of the website according to the size of the window is media query. When the window size shrinks or expands to a certain size, it will trigger the CSS to change the style of the webpage to fit the new window size.

I used JavaScript and jQuery to create the interactive header and the exhibition slideshow. When the user scrolls down a page, the header will move up and out of the screen so the user can concentrate on the main content. If the user wants to see the header again, they just need to scroll up. The slideshow in the main page allows the user to browse through different exhibitions without the need to move the page.
